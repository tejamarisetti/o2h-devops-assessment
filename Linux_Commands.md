# Linux Commands
## Display current user

```bash
whoami
```
## Show disk usage

```bash
df -h
```
## Show memory usage

```bash
free -m
```
## Find all .log files

```bash
find . -name "*.log"
```
## Create backup folder

```bash
mkdir backup
```
## Compress project directory

```bash
tar -czvf backup.tar.gz project/
```
